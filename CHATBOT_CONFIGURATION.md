Your chatbot now uses:

### 1. **Your Trained ML Models** (`train.py`)

### 3. **Gemini API - Complex Queries ONLY**
- ✅ Only used when confidence < 75%
- ✅ NEVER used for simple intents (greeting, thanks, product search, etc.)
- ✅ Used for truly complex, multi-turn conversations
- ✅ Grounded in your database (never invents products)

## 🎯 How It Works

### Query Flow:

```
User Message
    ↓
Load YOUR trained model (from train.py)
    ↓
Predict intent using YOUR dataset
    ↓
Confidence >= 75%? 
    ├─ YES → Use ML response from YOUR intents.json
    │        (Fast: ~300-500ms)
    │
    └─ NO → Use Gemini API for complex query
             (Slower: ~2-3s, but more intelligent)
```


## ⚙️ Configuration Options

### Adjust Confidence Threshold

Edit `api/chatbot_streaming.php`:

```php
// Line ~97
$highConfidenceThreshold = 0.75;  // Current: 75%

// More strict (use Gemini more often)
$highConfidenceThreshold = 0.85;  // 85%

// Less strict (use ML more often)
$highConfidenceThreshold = 0.65;  // 65%
```

Edit `api/chatbot_streaming.php`:

```php
// Line ~62
$preferredModels = [
    'mlp_neural_network.pkl',  // #1 choice
```

## 🎓 Why This Approach?

### Benefits:

1. **Uses Your Data** ✅
   - All responses from YOUR intents.json files
   - No external hardcoded responses
   - Fully customizable via your dataset

2. **Fast Response Times** ✅
   - Simple queries: ~300-500ms (ML only)
   - Complex queries: ~2-3s (Gemini, but worth it)
   - Models cached in memory

3. **Intelligent Routing** ✅
   - ML handles what it's good at (pattern matching)
   - Gemini handles what it's good at (context, reasoning)
   - Best of both worlds

4. **Scalable** ✅
   - Can add more intents to JSON files
   - Retrain anytime with `train.py`
   - No code changes needed

## 🔍 Troubleshooting

### Issue: "No trained model found"

**Solution:**
```bash
cd chatbot-ml
python train.py
```

Make sure models are created:
```
dir chatbot-ml\models\*.pkl
```

### Issue: Always using Gemini (even for simple queries)

**Cause:** Confidence threshold too high

**Fix:**
```php
// Lower threshold
$highConfidenceThreshold = 0.65;  // Was 0.75
```

### Issue: Never using Gemini (even for complex queries)

**Cause:** Confidence threshold too low OR model overfitting

**Fix:**
```php
// Raise threshold
$highConfidenceThreshold = 0.85;  // Was 0.75

// OR retrain with more diverse data
python train.py
```
